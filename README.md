# ポイントでしょうか

1からlen(A)+2までのすべての数値を含むセットと、リストAのセットを作成します。
そして、これら2つのセットの差集合を取り、リストAに存在しない最初の正の整数を返します。この方法では、リストの長さに関係なく、一貫した速度で結果を得ることができます。ただし、このコードは追加のメモリを使用します。<br>
※メモリ使用量が問題になる場合は、別のアプローチを検討する必要があります。
